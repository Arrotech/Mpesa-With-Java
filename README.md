## MPESA API's INTEGRATION

`Consume` MPESA API's with `Java`.
Such  API's include:

1. B2C API
2. B2B API
3. C2B API
4. Account Balance API
5. Transaction Status
6. Reversal
7. Lipa na M-Pesa Online Payment
8. Lipa na M-Pesa Online Query Request
